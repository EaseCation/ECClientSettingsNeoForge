# Java Edition Window Appearance PyRPC

`ECClientSettingsNeoForge` consumes window appearance commands from the shared
NeteaseBridge `floodgate:netease` dispatcher. This protocol is only for Java
Edition clients.

## Routing

All commands use these exact identifiers:

```text
namespace = ECJavaEditionClientMod
system    = ECClientWindowSystem
```

Supported S2C events are `SetWindowAppearanceEvent` and
`ResetWindowAppearanceEvent`. Protocol version 1 is one-way: the client does
not send a capability or result event.

## SetWindowAppearanceEvent

The `title` and `frame` members are independently optional. An omitted member
keeps its current state, while a command that omits both is invalid.

```json
{
  "version": 1,
  "title": {
    "mode": "custom",
    "text": "EaseCation"
  },
  "frame": {
    "mode": "rgb",
    "background": 1579032,
    "text": 16777215,
    "fallback": "dark"
  }
}
```

Title modes:

- `custom`: requires `text`, limited to 128 Unicode code points.
- `default`: restores the title generated by Minecraft and other installed mods.

Frame modes:

- `rgb`: requires an `background` integer in `0x000000..0xFFFFFF` and a
  `fallback` of `dark` or `system`. `text` is an optional RGB integer; the
  client chooses black or white for contrast when it is omitted.
- `dark`: requests the Windows dark title bar. Unsupported systems keep their
  system appearance.
- `system`: restores the native title bar values captured before the first
  server override.

Exact RGB title bar colors require Windows 11. On unsupported Windows versions,
an RGB request first rolls back any partial native change, then applies the
explicit `fallback`. Non-Windows clients still apply title commands and safely
ignore native frame commands.

### Paper API example

```java
NeteaseBridgeApi.api().notifyToClient(
        player.getUniqueId(),
        "ECJavaEditionClientMod",
        "ECClientWindowSystem",
        "SetWindowAppearanceEvent",
        Map.of(
                "version", 1,
                "title", Map.of(
                        "mode", "custom",
                        "text", "EaseCation"
                ),
                "frame", Map.of(
                        "mode", "rgb",
                        "background", 0x181818,
                        "text", 0xFFFFFF,
                        "fallback", "dark"
                )
        )
);
```

### Raw PyRPC shape

```text
[
  "ModEventS2C",
  [
    "ECJavaEditionClientMod",
    "ECClientWindowSystem",
    "SetWindowAppearanceEvent",
    {
      "version": 1,
      "title": {"mode": "custom", "text": "EaseCation"},
      "frame": {
        "mode": "rgb",
        "background": 1579032,
        "text": 16777215,
        "fallback": "dark"
      }
    }
  ],
  null
]
```

## ResetWindowAppearanceEvent

Reset both the title and native frame appearance with:

```json
{"version": 1}
```

The client also performs the same reset when the NeteaseBridge connection is
closed, or separately when the player disables either permission in EaseCation
Client Settings. Dimension changes and proxy backend transfers do not reset the
appearance while the Java server connection remains active.
